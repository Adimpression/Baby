Welcome to Like Places!
 
Who says there aren't interesting places to hang around?
 
We were losing touch with distanced friends and family
We tried to keep up via social networks, but updates faded because we rarely met them
Some we couldn't meet when we had the chance
Some we carelessly ignored
Some, we just couldn't create that moment to spend with them, be it dining out or a road trip
It just wasn't happening
We realized we just didn't find the place and time to do this
So, we created I Like Places!
 
If you've got people you miss and time to hang out, we've got the places for you!

Yes, indeed! The places you mark go into the main map where others can find it.
"Oh! are my places so public?"
Quite the contrary. Usually we show a places name based on an average of 10 name entries given to it.
This means private places like homes are unlikely to show. With time, we'll increse this quota for more safety.
You can mark any public interesting place and have moments there.
